# Python Revision
- to run python script `python <filename>` or `python3 <filename>`

## Quick keys:
- open and close terminal `ctrl + ~`
- clearing /cleaning terminal: type `clear` or `ctrl + l` 
- to view previous terminal commands cycle `up-arrow` or `down arrow`
- to toggle a comment `ctrl + ?`
- 3 single or three double quotes for multi-line comments

##Built-in functions:
- `print()` - prints values to to the console
- `input()` - opposite of print. gets input from user
